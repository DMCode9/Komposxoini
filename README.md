# Ψηφιακό Κομποσχοίνι (Digital Komboskini)

Ένα ψηφιακό κομποσχοίνι με τη μορφή Progressive Web App (PWA).

## Χαρακτηριστικά

- **Πολλαπλά Θέματα (Themes):** Σκοτεινό (Dark), Φωτεινό (Light), Βυζαντινό (Byzantine) και Μοναστηριακό (Monastic).
- **Progressive Web App (PWA):** Μπορεί να εγκατασταθεί στην αρχική οθόνη του κινητού σας (iOS / Android) ή του υπολογιστή σας και λειτουργεί εκτός σύνδεσης (offline).
- **Απλός Σχεδιασμός:** Καθαρό UI, βελτιστοποιημένο για γρήγορη και απρόσκοπτη χρήση.

## Εγκατάσταση και Χρήση

Δεδομένου ότι είναι μια εφαρμογή ιστού (Web App), δεν χρειάζεται να εγκαταστήσετε κάτι στο σύστημά σας. 
Για εγκατάσταση στο κινητό: Ανοίξτε την εφαρμογή στον browser και επιλέξτε "Προσθήκη στην αρχική οθόνη" (Add to Home Screen).

## Τεχνολογίες

Το project έχει υλοποιηθεί με κλασικές τεχνολογίες ιστού (Vanilla Web Technologies), χωρίς εξαρτήσεις (dependencies) από εξωτερικές βιβλιοθήκες:
- HTML5
- CSS3
- JavaScript (Vanilla)
